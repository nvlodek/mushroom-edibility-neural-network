# 🧠 Mushroom Edibility Classification using Neural Networks

## Overview
This project applies a neural network to classify mushrooms as edible or poisonous using 76 input features. It demonstrates how deep learning models can handle high-dimensional structured data for real-world classification tasks.

## Problem
Accurately identifying mushroom edibility is critical, as misclassification can have serious consequences. This project explores how neural networks can learn complex feature relationships to make reliable predictions.

## Approach
- Built a binary classification model using TensorFlow and Keras  
- Preprocessed a dataset with 76 categorical and numerical features  
- Split data into training and testing sets  
- Trained and tuned the model to improve performance  
- Evaluated results using accuracy and performance visualizations  

## Tools & Technologies
- Python  
- TensorFlow / Keras  
- scikit-learn  
- NumPy  
- matplotlib  

## Results
- Achieved high classification accuracy on test data  
- Visualized training vs. validation performance to monitor overfitting  
- Demonstrated strong performance on high-dimensional structured data  

## Feature Importance & Model Optimization
- Trained models using individual features to evaluate feature importance  
- Compared validation accuracy across features using bar chart visualization  
- Iteratively removed low-impact features and retrained models  
- Applied model checkpointing to preserve best-performing models  
- Analyzed performance differences between full-feature and reduced-feature models  

## Key Takeaways
- Neural networks effectively model complex relationships in structured datasets  
- Feature selection plays an important role in model performance  
- Visualization is essential for diagnosing overfitting and model behavior  

## How to Run
1. Clone the repository  
2. Install required libraries (`tensorflow`, `scikit-learn`, etc.)  
3. Run the main script or notebook
